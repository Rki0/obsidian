# Chapter 02. 수식어
## 수식어 거품
### 수식어 거품이 될 수 있는 것
수식어 : 문장에 없어도 되는 부가적인 성분
#### 전치사구
- 전치사 + 목적어 = 전치사구
- at / on / in / from
- Such as / in addition to
- plus / following / like / unlike
```
(At) the conference, we attended some lectures.
```
#### To 부정사구
To 부정사의 역할 : 명사, 형용사, 부사
##### 형용사의 경우
```
Somebody to love
Something to drink
```
