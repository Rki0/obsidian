## 동명사(동사 + ing)는 명사의 역할을 한다.
1. ___ + 동사 => 주어
	1. `단수 취급`하는 것을 문제로 많이 출제함.
2. 타동사 + ___ => 목적어
3. 전치사 + ___ => 목적어
4. be + ___ => 보어
## 동명사의 특징
| 자동사 | 타동사 | Be 동사 |
| ---- | ---- | ---- |
| Speak | Buying | be |
| Speaking quickly | Buying books | Being silent |
- 자동사 + 부사
- 타동사 + 목적어 / 목적 보어
- Be 동사 + 보어
- 능동과 수동 모두 가능
- 즉, 동사의 특징을 가지면서 명사의 역할을 한다.
## 동명사의 역할, 형태, 의미상 주어
### 동명사는 명사의 역할을 한다.
```
((Leasing) additional office space) will enable~
=> 동명사[구]가 주어가 되면 동사 단수 취급

He likes ((reading) novels).
=> 동사의 목적어

The documentary aims at ((promoting) alternative energy sources).
=> 전치사의 목적어

John's hobby is ((listening) to music).
=> 보어
```
### 동사는 동명사 자리에 올 수 없다.
```
Sarah is studying about [work / (working)] with animals in her zoology course.
```
### 수동형 : 'being p.p', 완료형 : 'having p.p'
```
My kids really like (being given) gifts. => 수동
I am sorry for (having told) a lie. => 완료
```
### 동명사의 의미상의 주어 : '소유격'
```
I don't enjoy ((Kevin's) singing).
((Her) attending) the conference was confirmed.
=> attending the conference가 주어, Her는 행위 주체
=> 즉, was는 동명사[구] 때문에 나오는거지 Her 때문이 아님.
=> 선지에는 [She / Her]로 헷갈리게 출제함
```
## 동명사를 목적어로 갖는 동사
### 동명사를 목적어로 갖는 동사들이 있다.
| 의미 | 단어 |
| ---- | ---- |
| 즐기다 | enjoy |
| 제안, 고려 | suggest / recommend / consider |
| 중지, 연기 | finish / quit / discontinue / give up / postpone |
| 부정적 | dislike / deny / mind / avoid |
```
My manager will (consider) [to give / (giving)] me a week's vacation.
```
### 동명사와 to 부정사를 모두 목적어로 갖는 동사들이 있다.
| 의미상의 차이 X |
| ---- |
| 감정 : like / love / hate / prefer |
| ⭐️시작 & 계속 : start / begin / continue |
| 시도 & 의도 : attempt / intend |
```
Let's (begin to review) the report.
Let's (begin reviewing) the report.
```

| 의미상의 차이 O |  |  |
| ---- | ---- | ---- |
| 동사 | 동명사 | To 부정사 |
| remember | 과거 | 미래 |
| forget | 과거 | 미래 |
| regret | ~한 것을 후회 | ~하게 되어 유감 |
```
I remember (to call) you.(미래)
I remember (calling) you.(과거)

I regret (telling) a lie.(후회)
I regret (to inform) you that ~ (유감)
```
## 동명사 vs 명사
### 동명사는 목적어 가능, 명사는 불가능
```
The corporation failed in [establishment / (establishing)] (a branch office) in Russia.
```
### 동명사 앞에는 부정관사 불가, 명사는 가능
- `an opening`은 예외!
- `the`를 붙이는 것은 가능하다.
```
Farmers noticed (a) significant [decreasing / (decrease)] in the size of their crops this year.
```
### 전치사 목적어
- 전치사 + ___ + 명사 => 해석으로 답을 찾아야한다.
	1. 형용사 / 과거분사 => 상태, 상황, 특징
	2. 동명사 => 동작을 설명
```
by ___ money

1. considerable 엄청난 자금에 의해서
2. considering 자금을 고려함으로써
```

- 전치사 + ___ + 한정사 + 명사 => 동명사만 가능
	1. 한정사 => 관사 / 지시형용사 / 소유격 / 수량형용사
	2. 명사 / 형용사는 불가 : great a movie(X) => a great movie(O)
	3. ⭐️동명사는 목적어로 "한정사 + 명사"
```
The company is in the process (of) ___ (a) new regulation.
(A) established (B) establishment ((C) establishing)
```
### 명사와 동명사의 의미가 다른 경우
| 명사(가산) | 동명사(불가산) |
| ---- | ---- |
| process 과정 | processing 처리 |
| plan 계획 | planning 기획 |
| seat 좌석 | seating 좌석배치 |
| fund 기금 | funding 자금지원 |
| price 가격 | pricing 가격책정 |
```
a marketing plan => 마케팅 계획(안)
planning department => 기획 부서

To ensure speedy [process / (processing)] of orders, make sure to provide accurate billing information.
```
## 동명사 관련 표현
### 동명사구 관용 표현
- be busy ~ing : ~하느라 바쁘다
- be worth ~ing : ~할 가치가 있다
- keep (on) ~ing : 계속해서 ~하다
- spend + 시간/돈 + ~ing : 시간/돈을 ~하는데 사용하다
- have [difficulty / trouble / a problem] (in) ~ing : ~하는데 어려움을 겪다
```
He (is busy editing) the article.
Mr.Wentworth (has difficulty seeing) without his glasses.
```
### 전치사 to + 동명사
- lead to : ~의 원인이 되다
- contribute to : ~에 공헌하다
- object to : 반대하다 = oppose
- be contributed / dedicated / committed to : ~에 헌신하다
- look forward to ~ing : ~하기를 고대하다
- be used to + 동사 원형 : 주어가 ~하는데 사용되다
- be used to + 명사 / 동명사 : 주어가 ~하는데 익숙하다
- used to + 동사 원형 : 과거에 ~하곤 했다
```
Solartech (is dedicated to) (developing) clean and renewable energy sources.
The film director (is committed to) (making) the best movie he can.
```
# 어휘
## Chapter 04. 부사 2
### otherwise : 달리, 그렇지 않으면
```
Subscribers will continue to receive these weekly newsletters unless (otherwise) requested.
달리 요청하지 않는 한 구독자들은 이 주간 소식지를 계속해서 받게 될 것입니다.
```
#### ⭐️ 출제 포인트
- unless otherwise requested : 달리 요청 받지 않는다면
- unless otherwise mentioned : 달리 언급되어 있지 않는다면
- the survey [indicated / say / think] otherwise : 그렇지 않게 나타났다
### fully : 충분히, 완전히
```
The two firms (fully) comprehend the difficulties involved in merging their companies.
두 회사는 그들의 회사를 합병하는 것에 관련된 어려움들을 충분히 이해한다.
```
#### ⭐️ 출제 포인트
- fully-furnished / equipped : 완벽하게 시설이 갖추어진
- fully aware of : 완벽하게 알고 있는
### significantly : 상당히
```
The number of government grants for university students has (significantly) increased.
대학생들을 위한 정부 보조금의 수는 상당히 증가해 왔다.
```
#### ⭐️ 출제 포인트
- significantly increase / decrease / improve
### currently : 현재
```
The new exhibit at the gallery is (currently) open to the public.
그 미술관에서의 새로운 전시회는 현재 대중에게 공개되어 있다.
```
#### ⭐️ 출제 포인트
1. 현재 진행시제 [is / are / am + ing]와 자주 사용
2.  = presently, at the moment, now
### occasionally : 때때로, 가끔
```
The CEO (occasionally) travels to East Asia to look for new investment opportunities.
최고 경영자는 새로운 투자 기회들을 찾기 위해 때때로 동아시아로 여행을 간다.
```
#### ⭐️ 출제 포인트
1. sometimes = sporadically
2. occasion (명) 때, 경우
3. occasional (형) 가끔
### promptly : 즉시, 정각에
```
The presentation by the design team began (promptly) after lunch.
설계팀에 의한 발표는 점심 이후 즉시 시작되었다.
```
#### ⭐️ 출제 포인트
- The show will start(begin) at 7:00 promptly. 정각에 = precisely
- Please have it delivered promptly. 신속하게 = quickly
