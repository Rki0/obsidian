```
SNS stands for Social Network Service.

stands for : ~를 의미하다.
```

영어로는 `Social media` 라고 한다.
프로필은 프로파일이라는 발음이 올바른 발음이다.

```
SNS 하세요?(프로필이 있어요?)
Are you on social media?

프로필이 없어도, SNS를 사용하는지 묻는다면...
Do you use social media?
이들은 눈팅족(Social Media Lurker || Lurker)이다.
Are you a social media lurker?

특정 SNS를 하는지 바로 묻기도 한다.
___ 하세요?
Are you on / Do you use (Facebook / Instagram / Twitter)?
```

```
SNS 해요.
I'm on / I use social media.

SNS 안해요.
I'm not on / I don't use social media.
```

```
A : Do you use social media?(SNS 하세요?)
B : I'm not on social media, but I'm a social media lurker.(SNS 프로필이 없는데, 눈팅족이에요.)
```
