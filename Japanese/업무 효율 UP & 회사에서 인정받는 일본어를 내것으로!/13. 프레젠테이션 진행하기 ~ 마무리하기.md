# PT 진행하기

> 슬라이드를 연결하는 표현은 지난 수업의 목차 순서 말하는 것과 동일하다.

```
먼저 처음으로
(まず)最初(さいしょ)に/まずはじめに

다음으로
次(つぎ)に

마지막으로
最後(さいご)に
```

> 그런데 `次(つぎ)に`와 `最後(さいご)に` 사이에 슬라이드가 너무 많을 경우에는
> 계속 `次(つぎ)に`라고 표현하는 것보다 아래 단어들도 사용해주면 좋다.

```
계속해서
続(つづ)いて、続(つづ)きまして
```

> 무언가 포인트가 되는 내용을 설명할 때에는 다음과 같이 말한다.

```
여기서 가장 주목해야할 부분은
ここで最(もっと)も注目(ちゅうもく)すべきところは、

여기서 가장 중요한 부분은
ここで最(もっと)も大事(だいじ)なところは、
```

> 무언가의 장점과 단점을 말할 때에는

```
장점
メリット

단점
デメリット
```

 >보통 장단점은 여러 개가 있기 때문에 장점, 단점을 말하기 전에 전체적으로 몇 가지인지를 설명하는 것이 좋다.
 
```
OO 서비스의 장점은 3개 있습니다. 첫번째..., 두번째..., 세번째...

OOサービスのメリットは3(みっ)つあります。一(ひと)つ目。。。、二(ふた)つ目。。。、三(みっ)つ目
```

> 만약에 두개가 있다면

```
OO 서비스의 단점은 두가지가 있습니다. 하나는..., 다른 하나는...

OO サービスのデメリットは二つあります。一(ひと)つは。。。、もう一(ひと)つは。。。
```

> 자료를 인용해서 말할 때는 다음과 같은 표현을 쓴다.

```
~에 의하면

~によりますと/によると
```

```
~의 조사에 따르면, OO 서비스의 매출은 매년 증가 경향에 있는 것을 알 수 있습니다.

~の調査(ちょうさ)によりますと、OOサービスの売上(うりあげ)は毎年(まいとし)増加(ぞうか)傾向(けいこう)にあることがわかります。
```

>프레젠테이션에는 다양한 그래프를 사용하기도 하는데,
>그래프를 표현하는 방법은 다음과 같다.

```
이 그래프를 봐주세요.
こちらのグラフをご覧(らん)ください。

이는 OO서비스의 1년 간의 매출을 나타내는 그래프입니다.
こちらは、OOサービスの1年(ねん)間(かん)の売上(うりあげ)を示(しめ)すグラフです。
```

> 막대 그래프나 선 그래프를 설명할 때 주로 쓰는 표현은 다음과 같다.

- 위로 올라가는 상승 곡선, 우상향 ↗️
	- 右(みぎ)肩(かた)上(あ)がり
- 아래로 내려가는 하강 곡선, 우하향 ↘️
	- 右(みぎ)肩(かた)さがり
- 변화가 별로 없는 직선 ➡️
	- 横(よこ)ばい

```
매출과 이용자 수 모두 우상향 성장을 보여주고 있습니다.

売上(うりあげ)と利用(りよう)者(しゃ)数(すう)ともに右(みぎ)肩(かた)上(あ)がりの成長(せいちょう)を見(み)せています。
```

> 원 그래프를 설명할 때 주로 쓰는 표현은 다음과 같다.

```
보시는 대로 "사용하기 쉽다"가 전체의 10%의 점유율을 차지하고 있습니다.

ご覧(らん)の通(とお)り、「使(つか)いやすい」が全体(ぜんたい)の10％のシェアを占(し)めています。
```
# PT 마무리하기

> 프레젠테이션을 마무리할 때는 다음과 같은 표현을 사용한다.

```
이상입니다. 들어주셔서 감사합니다.

こちらで以上(いじょう)となります。ご清聴(せいちょう)いただきありがとうございました。
```

> 그리고나서 질의응답 시간을 갖는다.

```
그럼, 질의응답 시간을 갖도록 하겠습니다. 

それでは、質疑応答(しつぎおうとう)に移(うつ)りたいと思(おも)います。
```

```
질문이 있으신 분은 손을 들어주십시오.
ご質問(しつもん)のある方(かた)は挙手(きょしゅ)をお願(ねが)いします。

질문이 있으신 분은 부탁드립니다.
ご質問(しつもん)のある方(かた)はお願(ねが)いいたします。

질문 등이 있으시면 부탁드립니다.
ご質問(しつもん)などございましたらお願(ねが)いいたします。
```

> 질문에 답할 때에는 다음과 같이 말한다.

```
질문 감사드립니다.

ご質問(しつもん)ありがとうございます。
```

> 상대방의 질문이 길어서 한번 요약을 해야될 경우에는 요약을 한 후에 다음과 같이 말해준다.
> 즉, 질문을 다시 확인하고 대답하는 것이 좋다.

```
~라는 것으로 괜찮습니까/맞나요?

~ということでよろしいでしょうか。
```

> 대답을 한 뒤에, 대답이 되셨나요~ 물어볼 때는 이렇게 말한다.

```
답이 되셨나요?

ご質問(しつもん)の答(こた)えになっていますか。
```

> 그 자리에서 바로 대답하기 어려운 질문이면 이렇게 말한다.

```
대단히 죄송합니다만, 그 점에 대해서는 지금 바로 대답해드릴 수 없으니
찾아보고 나중에 메일로 답을 드려도 될까요?

大変(たいへん)申(もう)し訳(わけ)ございませんが、その点(てん)については今(いま)すぐ回答(かいとう)しかねますので、
調(しら)べて後(のち)ほどメールにてお答(こた)えしてもよろしいでしょうか。
```
- ます형 + かねる
	- "~할 수 없다"로 비즈니스 신에서 많이 쓰는 정중한 표현

> 대외비 정보일 때는 다음과 같이 말한다.

```
대단히 죄송합니다만, 대외비 정보이므로 대답해드릴 수가 없습니다.

大変(たいへん)申(もう)し訳(わけ)ございませんが、社外秘(しゃがいひ)の情報(じょうほう)ですので、お答(こた)えすることができません。
```

> 이후 질의응답을 마무리 할 때는 다음과 같이 말한다.

```
이 이상 질문이 없으면 여기서 끝내도록 하겠습니다.
여러분, 긴 시간동안 감사했습니다.

これ以上(いじょう)質問(しつもん)がなければこちらで終(お)わりにしたいと思(おも)います。
みなさん、長(なが)い時間(じかん)ありがとうございました。
```
